HTML

Basics
Element Description
<html> The <html> tag marks the starting point of an HTML document, while </html> marks the end of the entire document
<head> Contains the metadata of the web page
<title> The title tag defines the title of a web page that appears in the browserâ€™s title bar, tab, and bookmarks
<body> The <body> tag in HTML is the primary container that holds all the visible content a user sees on a web page
<h1>to<h6> The <h1> tag is the highest heading level, while <h2> to <h6> is a subheading and can be used multiple times unlike the <h1>
<p> The <p> tag in HTML creates paragraphs on a web page, it helps structure and organize text content
<a> The <a> tag creates a hyperlink, also known as an anchor link, which can help you move to another web page or a specific section within the same page
<img> The <img> tag is an HTML element we can use to display an image on a web page
<ol> The <ol> tag is useful when you want to create an ordered list on a web page
<li> The <li> tag creates and separates items in a list
<ul> The <ul> tag is the list tag that arranges a group of items into a bulleted list
<div> The <div> tag in HTML defines a division or section within an HTML document
<span> The Span Tag, or the <span> tag in HTML, allows web designers to style specific parts of the text differently than the rest
<br> The Line Break or the <br> tag creates a line break or newline within a web page
<embed> to add GIFs and other files

Tables & Attributes
Element Description
<table> to create a table
<thead> set of rows defining the head of the columns of the table
<tbody> set of rows defining the data in the table
<tfoot> set of rows defining the summary of the table
<th> defines a cell as the header of a group of table cells
<tr> defines a row of cells in a table
<td> defines cells that contains data
scope="row" or "col" specifies whether a header cell is a header for a column, row, or group of columns or rows
colspan="number" used for an attribute that you want to merge two or more columns into a single column
rowspan="number" used if you want to merge two or more rows
Semantics
Element Description
<article> specifies independent, self-contained content
<aside> defines some content aside from the content it is placed in (like a sidebar)
<figcaption> defines a caption for a <figure> element
<figure> pecifies self-contained content, like illustrations, diagrams, photos, code listings, etc
<footer> defines a footer for a document or section
<header> represents a container for introductory content or a set of navigational links
<main> Specifies the main content of a document
<nav> defines a set of navigation links
<section> defines a section in a document for eg. chapters, introductions, contact information, etc
